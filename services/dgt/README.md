# DGT Service - Digital Gold Trader

Port: 3008 (local) | 3380-3389 (production)

## Quick Start

```bash
# Build
nx run dgt:build

# Run
nx run dgt:api

# Health check
curl http://localhost:3008/health

# API docs
open http://localhost:3008/api-docs
```
