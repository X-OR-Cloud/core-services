# AIVP Service - AI Video Processing

Port: 3007 (local) | 3370-3379 (production)

## Quick Start

```bash
# Build
nx run aivp:build

# Run
nx run aivp:api

# Health check
curl http://localhost:3007/health

# API docs
open http://localhost:3007/api-docs
```
